📌 About the Portal

The Community Event Portal helps residents explore, register, and participate in local city events such as music festivals, sports activities, workshops, exhibitions, and cultural programs.

📝 How to Register

  • Go to the Registration Form section
  • Enter your name and email address
  • Select an event and event date
  • Submit the form successfully
  • You will receive a confirmation message

✨ Portal Features

  • Event image gallery
  • Real-time feedback character counter
  • Promo event videos
  • Geolocation for nearby events
  • Local storage for saving preferences
  • Responsive mobile-friendly design

🌐 Browser Support

This portal works best on modern browsers like Chrome, Edge, Firefox, and Safari.
