# Local Community Event Portal

The Local Community Event Portal is a responsive and interactive web application developed to help residents discover, register for, and engage with local community events. The project demonstrates the practical implementation of HTML5, CSS3, JavaScript, and Bootstrap 5 concepts.

## Features

- Responsive Navigation Menu
- Event Gallery with Interactive Images
- Event Registration Form with Validation
- Real-Time Character Counter
- Event Search and Category Filtering
- Event Fee Display Based on Selection
- Geolocation API for Finding Nearby Events
- Local Storage and Session Storage Support
- Embedded Event Promo Video
- Dynamic Event Cards using JavaScript
- Bootstrap 5 Components (Navbar, Cards, Forms, Modal, Accordion)
- Mobile-Friendly Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Local Storage API
- Geolocation API
